# Projeto CRUD Java

Este projeto implementa um CRUD de Produtos com autenticação de usuários usando Java, Swing e MySQL.

## Funcionalidades:
- Login de usuário
- Cadastro de novo usuário
- CRUD de Produtos (inserção e listagem)

## Tecnologias:
- Java
- Swing
- MySQL
